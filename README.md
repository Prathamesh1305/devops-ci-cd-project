# CI/CD Pipeline using Jenkins, Docker & AWS

## Project Overview
This project demonstrates a complete DevOps CI/CD pipeline for automated build, test, and deployment of a containerized web application.

## Tools Used
- GitHub
- Jenkins
- Docker
- AWS EC2
- Linux
- Shell Script

## Workflow
1. Developer pushes code to GitHub  
2. Jenkins automatically triggers pipeline  
3. Docker image is built and pushed to Docker Hub  
4. Application is deployed on AWS EC2  

## Outcome
Fully automated deployment pipeline with zero manual intervention.
